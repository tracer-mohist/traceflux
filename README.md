# traceflux

**A text search engine with associative keyword discovery.**

**Philosophy**: UNIX philosophy — simple, composable, human-centric.

---

## Core Insight

Traditional search finds what you know.  
**traceflux** helps discover what you don't know to search for.

**"左脚踩右脚原地起飞"** — Lift off by stepping on each other's feet.

## Status

**Phase**: Design & Research  
**Not ready for production**

## Name Origin

**traceflux** (noun):
- **trace**: Follow traces, leave paths (Tracer's identity)
- **flux**: Flow, change, continuous movement

**Meaning**: Tracing the flow of information, discovering related terms like water currents.

**Chinese**: 迹流 (Jì Liú) — 痕迹流动，关联涌现
